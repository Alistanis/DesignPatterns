            <h1 class='name'>
              <span class='type'>class</span>
              Patterns::Observer
            </h1>
          </div>
          <section id='content'>
            <div id='text'>
              <div id='description'>

                <p><a href="Observer.html">Observer</a> -&gt; <a
                href="../Object.html">Object</a></p>

                <p>An <a href="Observer.html">Observer</a> class that receives notifications
                about events from subjects it is observing Ruby has an <a
                href="Observer.html">Observer</a> module built in to the language, but we
                will build one to demonstrate how it works and to give it flexibility that
                we want for this application This implementation can be inherited or used
                by itself. For example, it could be an <a href="Admin.html">Admin</a> class
                watching a <a href="User.html">User</a> class, or it could just be a class
                called UserObserver. We would simply add this class to whatever subject
                list we want to observe</p>
              </div>
